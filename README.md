# MoBRet
MoBret adalah platform streaming film berbasis web yang menyediakan fitur pencarian film, informasi detail, trailer, rating, ulasan pengguna, dan simulasi pemutaran film. Dengan desain bernuansa pastel yang modern serta integrasi REST API, MoBret menghadirkan pengalaman eksplorasi dan hiburan film yang interaktif, nyaman, dan mudah digunakan.
